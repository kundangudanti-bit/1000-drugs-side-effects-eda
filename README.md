## Overview
This project performs exploratory data analysis (EDA) on a healthcare dataset to
analyze drug usage, side effects, patient demographics, and treatment effectiveness.

## Dataset
The dataset contains information related to drugs, reported side effects, patient
demographics, and improvement metrics.

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas, NumPy
- Matplotlib, Seaborn

## Analysis Performed
- Data cleaning and preprocessing (missing values, duplicates, categorical standardization)
- Univariate, bivariate, and multivariate analysis
- Statistical summaries and data visualizations

## Key Insights
- Identified patterns between drugs and reported side effects
- Observed relationships influencing treatment effectiveness
- Highlighted trends across patient demographics

## Conclusion
The analysis provides meaningful insights into drug performance and side effects,
supporting data-driven healthcare analysis.
